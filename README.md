**Dezvoltare Aplicație IoT: Conectivitate BLE, Wi-Fi Provisioning și Integrare REST API**

* **Arhitectură și Conectivitate:** Am dezvoltat un firmware în C/C++ folosind mediul Arduino, configurând dispozitivul (bazat pe arhitectura ESP32) pentru a acționa ca un server Bluetooth Low Energy (BLE). Aplicația permite comunicarea bidirecțională cu clienți externi prin intermediul caracteristicilor și descriptorilor BLE.
* **Wi-Fi Provisioning prin BLE:** Am implementat un mecanism securizat de configurare a rețelei care procesează comenzi prin BLE pentru a scana rețelele Wi-Fi disponibile, returnând informații precum SSID, puterea semnalului (RSSI) și tipul de criptare. Sistemul realizează conectarea dinamică a dispozitivului la o rețea locală folosind credențialele primite de la distanță.
* **Interogare REST API și Gestionare HTTP:** Am utilizat clientul HTTP nativ pentru a efectua cereri GET către endpoint-uri REST externe, preluând structuri de date de pe servere web. Implementarea include un algoritm robust pentru gestionarea automată a redirect-urilor HTTP (HTTP 301, 302, 303), asigurând o extragere stabilă a datelor chiar și la schimbarea rutei.
* **Procesare și Management JSON:** Am utilizat biblioteca ArduinoJson pentru serializarea și deserializarea eficientă a pachetelor de date, gestionând alocarea memoriei direct pe microcontroler. Sistemul extrage date specifice din răspunsurile API complexe și le restructurează în noi obiecte JSON pentru a fi raportate utilizatorului.
* **Transmisia Datelor Optimizată (Chunking):** Am dezvoltat o funcție de fragmentare pentru a transmite secvențial payload-urile JSON mari înapoi către clientul BLE. Datele sunt împărțite în blocuri și transmise prin notificări BLE succesive, prevenind pierderea pachetelor și respectând limitările de dimensiune ale protocolului.
* **Tehnologii și Concepte:** C/C++, IoT (Internet of Things), Bluetooth Low Energy (BLE), Comunicații Wi-Fi, HTTPClient, REST API, Parsare JSON.
